# Spark - Social Media App
# Overview
This is a software web application implemented using ASP.Net Core, Identity, Entity Framework ORM and Microsoft SQL Express Server.

## Vision
Spark’s starting goal was to combine positive features derived from the ones present in dominant social media platforms (inspired majorly by Twitter and Tumblr) and balance them in order to bring forthwith the most appealing parts of each
element; essentially finding inspiration in blending the two aforementioned apps in such
a way to optimize their capabilities.

## Features
It offers two types of distinct posts, encompassing the appeal of both rapid-fire short
form content and longer, more detailed posts. It aims to encourage profile customization
and website personalization in order to facilitate users in creating a sense of identity,
ownership and comfort. Per-post privacy settings are not commonly encountered, but
upon consideration they maintain great appeal in controlling each and every post’s reach.
The repost/requote system was expanded to accommodate both types of posts and the
interaction system was outfitted with a save functionality as well. Concisely, Spark holds
the ambition of replicating the engaging gamified user interface portrayed by X(Twitter),
whilst making space and attempting to bring into play long-form posts as well and other
key aspects heavily inspired by Tumblr.
